### برای نصب در مسیر روت پروژه خود دستور زیر را در ریشه پروژه اجرا کنید 

``composer require miladimos/laravel-social``

### برای نصب فایل های پکیج از دستور زیر استفاده کنید 

``php artisan social:install``
